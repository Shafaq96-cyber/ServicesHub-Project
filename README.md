# ServicesHub-Project
## 🛠 Technology Stack

The following technologies were used to develop this full-stack application:

* Backend: [Node.js](https://nodejs.org/) - A JavaScript runtime built on Chrome's V8 JavaScript engine, used for handling server-side logic.
* Web Framework: [Express.js](https://expressjs.com/) - A fast, unopinionated, minimalist web framework for Node.js used for routing and API development.
* Database: [MySQL](https://www.mysql.com/) - A relational database management system used to store and manage data for users, services, and bookings.
* Frontend: Developed using **HTML5**, **CSS3**, and **Vanilla JavaScript** to create a responsive and interactive user interface.
* Version Control: [Git & GitHub](https://github.com/) - Used for source code management and project documentation.
